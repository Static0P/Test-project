# Momentum Lab

A polished single-page productivity app you can run instantly in any browser.

## Features

- 🎯 Focus sprint timer with 15 / 25 / 50 minute presets.
- ✅ Mission list with done/undo/delete controls.
- 💾 Task persistence via localStorage.
- 🚀 Random motivational "ignition" quotes.

## Run

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
